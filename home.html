What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents that represents the structure of a document as a tree of objects. Each element, attribute, and piece of text in the document is a node in this tree, allowing developers to manipulate the content and structure using languages like JavaScript. The DOM enables dynamic changes to the webpage in response to user interactions.

How do you select an element from the DOM?

To select an element from the DOM, you can use various methods provided by JavaScript. Commonly used methods include document.getElementById() for selecting by ID, document.getElementsByClassName() for class names, and document.querySelector() for selecting using CSS selectors. These methods allow you to access and manipulate HTML elements dynamically.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of individual listeners on multiple child elements. When an event occurs on a child, it bubbles up to the parent, which can handle the event. This approach is useful because it reduces memory usage, simplifies code management, and allows dynamically added elements to inherit the event listener without requiring additional code. It enhances performance and maintainability in applications with many interactive elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you can use JavaScript methods and properties: Attributes: Use element.getAttribute('attributeName') to retrieve an attribute's value. Use element.setAttribute('attributeName', 'value') to change or add an attribute. You can also directly access properties like element.src, element.href, etc., for common attributes. Styles: Use element.style.propertyName to change individual CSS properties (e.g., element.style.color = 'red';). To set multiple styles, you can use element.style.cssText = 'property1: value; property2: value;'. For class manipulation, use element.classList.add('className'), element.classList.remove('className'), or element.classList.toggle('className'). These methods allow you to dynamically adjust an element's behavior and appearance in response to user interactions or other events.
